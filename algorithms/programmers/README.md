### Bruteforcing

브루트포스 알고리즘(Bruteforcing)이란 **문제의 가능한 모든 해답 후보를 하나씩 체계적으로 검사하여 문제에 맞는 답을 찾는 알고리즘 기법**입니다. 모든 경우의 수를 탐색하기 때문에 100%의 정확성을 보장하지만, 가능한 해답의 수가 많으면 처리해야 할 경우의 수도 기하급수적으로 늘어나 높은 시간 복잡도를 가지게 됩니다.
